## ▶️ How to run

1. Clone repository
2. Run `npm install`
3. Run `npm run dev`

---

## 🧩 Description

A full-featured product catalog for phones, tablets, and accessories built as part of an educational React project.
The app is based on a Figma design (Original Dark theme) and includes catalog browsing, product details, shopping cart, and favorites — all with persistent state via localStorage.

---

## 🎯 Features

- 🏠 **Home page** — picture slider with auto-scroll, Hot Prices block, Brand New block, Shop by Category
- 📱 **Product pages** — separate pages for Phones, Tablets, Accessories with sorting, pagination, and search via URL params
- 🔍 **Product Details page** — image gallery, color & capacity selection, Tech Specs, About section, "You may also like" block, Breadcrumbs, Back button
- 🛒 **Shopping Cart** — add/remove items, change quantity, total calculation, checkout confirmation, saved to localStorage
- ❤️ **Favorites page** — add/remove favorites, counter in header, saved to localStorage
- 🔎 **Search** — debounced search saved in URL query params
- 📄 **Not Found page** — for all unknown routes
- 📱 **Fully responsive** — Mobile, Tablet, Desktop
- ✨ **Smooth hover effects** on all interactive elements
